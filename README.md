# Spring Boot Project Setup

Follow these steps to create your Spring Boot project:

## 1. Go to Spring Initializr
Open [https://start.spring.io](https://start.spring.io) in your browser.

## 2. Generate the Project
- Choose your preferred **Project**, **Language**, and **Spring Boot version**.
- Fill in **Group**, **Artifact**, and other project metadata.
- Select the dependencies you need (e.g., Spring Web, Spring Data JPA, etc.).
- Click **Generate** to download the project as a ZIP file.

## 3. Open in Your IDE
- Extract the downloaded ZIP file.
- Open the project in your preferred IDE (IntelliJ, Eclipse, VS Code, etc.).
- Build and run your application.

You are now ready to start developing your Spring Boot project!
